# Fitness Tracker Client

## Overview
The Fitness Tracker Client is a web application designed to help users manage their fitness journey. It allows users to book training sessions, make payments, and track their progress. The application is built using React, Tailwind CSS, and integrates with Stripe for payment processing.

## Key Features
- **User Authentication**: Secure login and registration using context API.
- **Trainer Booking**: Users can view available trainers and book sessions.
- **Payment Processing**: Integration with Stripe for secure payment handling.
- **Responsive Design**: Built with Tailwind CSS for a mobile-friendly experience.
- **Real-time Data Fetching**: Utilizes React Query for efficient data fetching and caching.
- **Toast Notifications**: User feedback through toast notifications for actions like successful bookings and payments.

## Technologies Used
- **React**: Frontend library for building user interfaces.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **React Router**: For routing and navigation.
- **Axios**: For making HTTP requests to the server.
- **Stripe**: For handling payments.
- **React Query**: For data fetching and state management.
- **Framer Motion**: For animations and transitions.
- **Firebase**: For authentication and real-time data.
## Usage
- Navigate to the application in your browser (usually at `https://fitness-tracker-project-96f15.web.app/`).
- Register or log in to your account.
- Browse trainers and book sessions.
- Complete payments through the Stripe integration.
